void GPS_getMsg ( void  )

Read the message from the USART buffer.

Reads the last character that has been read by the USART, and if it detects that is a GPS message, stores it to the buffer of GPS messages. If detect that the full message has been received, it will call the GPS_parseData() function to process it. Must be called from the USART IRQ.
