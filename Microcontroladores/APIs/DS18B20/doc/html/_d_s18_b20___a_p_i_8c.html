void DS18_convertTemp ( void  )

Send the convert temp to all DS18B20 sensors.

First sends a reset pulse to the sensors, then writes the skip ROM command and finally the convert command. Waits 750ms to complete the conversion. Uses the DS18_WriteByte() and DS18_Reset() functions internally.

DS18_DATA DS18_readSensor ( void  )

Read the temperature from the DS18B20 sensor.

Does the full process of reading the temperature from the sensor. First it sends the reset command to the sensor. After that, sends the skip rom and convert temp commands and waits to complete the conversion. Finally reads the scratchpad from the sensor and converts the values to float. Internally uses the DS18_Reset(), DS18_WriteByte and DS18_ReadData functions.

Returns
The temperature reading of the sensor.
