Read the temperature from the D\+S18\+B20 sensor. 

Does the full process of reading the temperature from the sensor. First it sends the reset command to the sensor. After that, sends the skip rom and convert temp commands and waits to complete the conversion. Finally reads the scratchpad from the sensor and converts the values to float. Internally uses the \hyperlink{_d_s18_b20___a_p_i_8c_a04e8c8d756559ea7dbc6a1895f207860}{D\+S18\+\_\+\+Reset()}, D\+S18\+\_\+\+Write\+Byte and D\+S18\+\_\+\+Read\+Data functions. \begin{DoxyReturn}{Returns}
