void SL_Init ( )

Stars the timer needed for the SL_sleepForSecs() function.

Starts the timer and sets the preescaler at the max setting. It also enables the TIM7 interrupt.

void SL_sleepForSecs ( uint16_t  secs)

Sleep the microcontroller for x seconds.

Puts the board in low power sleep mode and waits for the interrupt that will come from TIM7 after the time elapses.

Parameters
secsThe seconds that the board will be in sleep mode.
void TIM7_IRQHandler ( )

TIM7 IRQ Handler function.

Handles the interrupt generated by the TIM7. In this case, it just clears the interrupt.
